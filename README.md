# Konecta Internship Program - AI & ML Track

This repository showcases the projects and deliverables I completed during my Internship Program at [Konecta](https://konecta.com/) as part of the Artificial Intelligence & Machine Learning track.

## Tasks Overview

### Task 1: Data Cleaning & Preprocessing
Data cleaning exercise with cafe sales dataset - handling missing values, normalization, and feature engineering.

### Task 2: Web Scraping
Python web scraper using BeautifulSoup to collect e-commerce product data (100+ products).

### Task 3: Diabetes Classification
Supervised learning pipeline comparing multiple classifiers (SVM, Random Forest, XGBoost, etc.) with hyperparameter tuning.

### Task 4: Customer Segmentation
Unsupervised clustering (K-Means, Hierarchical, DBSCAN) for wholesale customer analysis.

### Task 5: Sports Image Classification
CNN-based deep learning model for sports image classification with hyperparameter optimization.  
**Demo:** [![Hugging Face](https://img.shields.io/badge/🤗%20Hugging%20Face-Space-blue?logo=huggingface)](https://huggingface.co/spaces/ahmed-ayman/image-classification-using-cnn)

### Task 6: RAG-Based PDF Chatbot
Retrieval-Augmented Generation chatbot with conversational memory for answering queries from product manuals.
