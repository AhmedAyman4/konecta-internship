### ERP Chatbot

A lightweight app for semantic querying of business documents (invoices, purchase orders, approvals) using vector search and an LLM-backed answer generator.

### Live demo & API
- Streamlit app (demo): https://huggingface.co/spaces/ahmed-ayman/erp-chatbot  
- API (document processor): https://huggingface.co/spaces/ahmed-ayman/erp_chatbot_api
